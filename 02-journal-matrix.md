## Primary journal notes

- `Cluster Computing` is the default target because its scope is closest to cluster capacity management.
- Use a content-first manuscript structure that does not hard-code a publisher-specific visual layout too early.
- Before submission day, re-check:
  - author instructions
  - allowed article type
  - AI disclosure language
  - reference style details
  - any current submission platform requirements

## Backup journal notes

- `Journal of Grid Computing` stays compatible with the same core narrative if the manuscript foregrounds distributed resource sharing and scaling.
- Keep section names and bibliography organization conventional so the transfer cost stays low.
